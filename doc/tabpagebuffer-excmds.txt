VARIABLES				*tabpagebuffer-excmds-variables*

				*g:tabpagebuffer#excmds#delete_keeptabpage*
g:tabpagebuffer#excmds#delete_keeptabpage
	If this variable is 1, |:TpbDelete|, |:TpbWipeout|, |:TpbUnload|,
	|:TpbDeleteAll|, |:TpbWipeoutAll|, |:TpbUnloadAll| keep current tab.

------------------------------------------------------------------------------
COMMANDS				*tabpagebuffer-excmds-commands*

:TpbFiles[!]					*TpbFiles* *TpbBuffers* *TpbLs*
:TpbBuffers[!]
:TpbLs[!]
	Show all buffers only in the current tab.
	Also see |:files||, |:buffers|, |:ls|.


:[N]TpbDelete[!]						*TpbDelete*
:TpbDelete[!] [N]
:TpbDelete[!] {bufname}
:TpbDelete[!] N1 N2 ...
:N,MTpbDelete[!]
	Delete buffer only in the current tab.
	Also see |:bdelete|.

:[N]TpbWipeout[!]						*TpbWipeout*
:TpbWipeout[!] [N]
:TpbWipeout[!] {bufname}
:TpbWipeout[!] N1 N2 ...
:N,MTpbWipeout[!]
	Really delete buffer only in the current tab.
	Also see |:bwipeout|.

:[N]TpbUnload[!]						*TpbUnload*
:TpbUnload[!] [N]
:TpbUnload[!] {bufname}
:TpbUnload[!] N1 N2 ...
:N,MTpbUnload[!]
	Unload buffer only in the current tab.
	Also see |:bunload|.


:TpbDeleteAll							*TpbDeleteAll*
	Delete all buffers only in the current tab.

:TpbWipeoutAll							*TpbWipeoutAll*
	Really delete all buffers only in the current tab.

:TpbUnloadAll							*TpbUnloadAll*
	Unload all buffers only in the current tab.


:[N]TpbBuffer[!] [N]						*TpbBuffer*
:TpbBuffer[!] {bufname}
	Edit buffer only in the current tab.
	Also see |:buffer|.

:[N]STpbBuffer [N]						*STpbBuffer*
:STpbBuffer {bufname}
	Split window and edit buffer only in the current tab.
	Also see |:sbuffer|.

:[N]VTpbBuffer [N]						*VTpbBuffer*
:VTpbBuffer {bufname}
	Split vertically and edit buffer only in the current tab.

:[N]TTpbBuffer [N]						*TTpbBuffer*
:TTpbBuffer {bufname}
	Open a new tab page and edit buffer only in the current tab.


:[N]TpbNext[!] [N]						*TpbNext*
	Go to [N]th next buffer in buffers only in the current tab.
	Also see |:bnext|.

:[N]TpbPrevious[!] [N]						*TpbPrevious*
	Go to [N]th previous buffer in buffers only in the current tab.
	Also see |:bprevious|.

:[N]TpbModified[!] [N]			*TpbModified* *TpbModifiedNext*
:[N]TpbModifiedNext[!] [N]
	Go to [N]th next modified buffer only in the current tab.
	Also see |:bmodified|.

:[N]TpbModifiedPrevious[!] [N]				*TpbModifiedPrevious*
	Go to [N]th previous modified buffer only in the current tab.


:[N]STpbNext [N]						*STpbNext*
	Split window and go to [N]th next buffer in buffers only in the
	current tab.
	Also see |:sbnext|.

:[N]STpbPrevious [N]						*STpbPrevious*
	Split window and go to [N]th previous buffer in buffers only in the
	current tab.
	Also see |:sbprevious|.

:[N]STpbModified [N]			*STpbModified* *STpbModifiedNext*
:[N]STpbModifiedNext [N]
	Split window and go to [N]th next modified buffer in buffers only in
	the current tab.
	Also see |:sbmodified|.

:[N]STpbModifiedPrevious [N]				*STpbModifiedPrevious*
	Split window and go to [N]th previous modified buffer in buffers only
	in the current tab.


:[N]VTpbNext [N]						*VTpbNext*
	Split vertically and go to [N]th next buffer in buffers only in the
	current tab.

:[N]VTpbPrevious [N]						*VTpbPrevious*
	Split vertically and go to [N]th previous buffer in buffers only in the
	current tab.

:[N]VTpbModified [N]			*VTpbModified* *VTpbModifiedNext*
:[N]VTpbModifiedNext [N]
	Split vertically and go to [N]th next modified buffer in buffers only in
	the current tab.

:[N]VTpbModifiedPrevious [N]				*VTpbModifiedPrevious*
	Split vertically and go to [N]th previous modified buffer in buffers only
	in the current tab.


:[N]TTpbNext [N]						*TTpbNext*
	Open a new tab page and go to [N]th next buffer in buffers only in the
	current tab.

:[N]TTpbPrevious [N]						*TTpbPrevious*
	Open a new tab page and go to [N]th previous buffer in buffers only in the
	current tab.

:[N]TTpbModified [N]			*TTpbModified* *TTpbModifiedNext*
:[N]TTpbModifiedNext [N]
	Open a new tab page and go to [N]th next modified buffer in buffers only in
	the current tab.

:[N]TTpbModifiedPrevious [N]				*TTpbModifiedPrevious*
	Open a new tab page and go to [N]th previous modified buffer in buffers only
	in the current tab.


:TpbRewind[!]						*TpbRewind* *TpbFirst*
:TpbFirst[!]
	Go to first buffer only in the current tab.
	Also see |:brewind|, |:bfirst|.

:TpbLast[!]							*TpbLast*
	Go to last buffer only in the current tab.
	Also see |:blast|.

:TpbModifiedRewind[!]			*TpbModifiedRewind* *TpbModifiedFirst*
:TpbModifiedFirst[!]
	Go to first modified buffer only in the current tab.

:TpbModifiedLast[!]					*TpbModifiedLast*
	Go to last modified buffer only in the current tab.


:STpbRewind					*STpbRewind* *STpbFirst*
:STpbFirst
	Split window and go to first buffer only in the current tab.
	Also see |:sbrewind|, |:sbfirst|.

:STpbLast							*STpbLast*
	Split window and go to last buffer only in the current tab.
	Also see |:sblast|.

:STpbModifiedRewind			*STpbModifiedRewind* *STpbModifiedFirst*
:STpbModifiedFirst
	Split window and go to first modified buffer only in the current tab.

:STpbModifiedLast					*STpbModifiedLast*
	Split window and go to last modified buffer only in the current tab.


:VTpbRewind					*VTpbRewind* *VTpbFirst*
:VTpbFirst
	Split vertically and go to first buffer only in the current tab.

:VTpbLast							*VTpbLast*
	Split vertically and go to last buffer only in the current tab.

:VTpbModifiedRewind			*VTpbModifiedRewind* *VTpbModifiedFirst*
:VTpbModifiedFirst
	Split vertically and go to first modified buffer only in the current
	tab.

:VTpbModifiedLast					*VTpbModifiedLast*
	Split vertically and go to last modified buffer only in the current
	tab.


:TTpbRewind					*TTpbRewind* *TTpbFirst*
:TTpbFirst
	Open a new tab page and go to first buffer only in the current tab.

:TTpbLast							*TTpbLast*
	Open a new tab page and go to last buffer only in the current tab.

:TTpbModifiedRewind			*TTpbModifiedRewind* *TTpbModifiedFirst*
:TTpbModifiedFirst
	Open a new tab page and go to first modified buffer only in the
	current tab.

:TTpbModifiedLast					*TTpbModifiedLast*
	Open a new tab page and go to last modified buffer only in the current
	tab.


:[N]TpbUnhide [N]				*TpbUnhide* *STpbUnhide*
:[N]STpbUnhide [N]
	Rearrange the screen to open one window for each loaded buffer only in
	the current tab.
	Also see |:unhide|, |:sunhide|.

:[N]VTpbUnhide [N]						*VTpbUnhide*
	Rearrange the screen to open one vertically window for each loaded
	buffer only in the current tab.

:[N]TTpbUnhide [N]						*TTpbUnhide*
	Rearrange the screen to open one tab for each loaded buffer only in
	the current tab.


:[N]TpbAll [N]						*TpbAll* *STpbAll*
:[N]STpbAll [N]
	Rearrange the screen to open one window for each buffer only in the
	current tab.
	Also see |:unhide|, |:sunhide|.

:[N]VTpbAll [N]							*VTpbAll*
	Rearrange the screen to open one vertically window for each buffer
	only in the current tab.

:[N]TTpbAll [N]							*TTpbAll*
	Rearrange the screen to open one tab for each buffer only in the
	current tab.
